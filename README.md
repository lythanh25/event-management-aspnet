# Event Management System (ASP.NET Web Forms)

## 📌 Description
Internal system for managing company events

## 🚀 Technologies
- ASP.NET Web Forms
- C#
- SQL Server

## ✨ Features
- Create and manage events
- Register participants
- Approve requests
- User management
